Petunjuk

Anda juga dapat mengetik menggunakan keyboar komputer:

  • Huruf dengan variasi titik dapat dijelajahi dengan tombol apostrop: t', H', d', s', S', T', g', j'
  • Bila menggunakan keyboard AR (Arab) tombol apostrop biasanya terletak pada tombol shift+M
  • Unicode Harakat pepet masih dalam proses pengajuan.
  • Untuk pengetikan angka, lebih disarankan menggunakan virtual keyboard
